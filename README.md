## 🚀 Giới Thiệu

Repository này là **kho lưu trữ toàn bộ bài tập về nhà** của học phần **Lập trình hướng đối tượng OOP** — được tổ chức khoa học theo từng tuần, từng bài.

Từ những bước đi đầu tiên với biến và vòng lặp, đến việc xây dựng một hệ thống quản lý sản phẩm hoàn chỉnh với **validation**, **khuyến mãi**, **phân loại danh mục**, và **báo cáo doanh thu** — tất cả đều được ghi lại tại đây.

## 🧠 Các Khái Niệm OOP Đã Áp Dụng

```
✔ Encapsulation   — private fields + getter/setter
✔ Constructor     — overloading (no-arg, 2-arg, 3-arg)
✔ Static Members  — counter, totalProducts, totalRevenue
✔ Composition     — Product HAS-A Category
✔ Validation      — tách logic vào lớp tiện ích riêng
✔ Method Chaining — constructor delegation (this())
```

## ⚡ Hướng Dẫn Chạy Nhanh

```bash
# 1. Clone repository
git clone https://github.com/<your-username>/baitapvenha.git

# 2. Vào thư mục bài 5 tuần 3
cd baitapvenha/Tuan3/Bai5/src

# 3. Biên dịch
javac -d out model/Category.java model/Product.java util/ProductValidator.java Main.java

# 4. Chạy
java -cp out Main
```
